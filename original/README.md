## Como executar o projeto

```bash
# Clonar o repositório
git clone https://github.com/gustavo-wanderley/proffy.git

# Entrar no diretório
cd proffy/original

# Baixar as dependências
npm i

# Executar o servidor
npm run dev
```

Feito isso, abra o seu navegador e acesse `http://localhost:8080/`
